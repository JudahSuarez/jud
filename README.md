# Live Object Detection App

This is a simple Streamlit web app that uses YOLOv8 to detect objects in real time using a webcam. The app can detect objects such as persons, cell phones, and bottles. It also saves detected frames when a person or bottle appears on the camera.

## Features

- Real-time webcam object detection
- YOLOv8 model integration
- Adjustable confidence threshold
- Object count display
- Warning message when a cell phone is detected
- Automatically saves frames when a person or bottle is detected
